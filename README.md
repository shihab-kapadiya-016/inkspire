# 🖋️ Inkspire

Inkspire is a modern full-stack blogging platform built with the MERN stack (MongoDB, Express.js, React, Node.js). Designed to be clean, responsive, and user-focused, it allows users to read, write, and interact with blog posts effortlessly.

---

## 🚀 Features

- ✍️ Create, edit, and delete blog posts
- 💬 Comment system with like/dislike functionality
- 👤 User registration, login, and profile management
- 📰 Responsive frontend using React and Tailwind CSS
- 🔐 JWT-based authentication and protected routes
- 📦 Backend API with Express and MongoDB
- 📸 Image uploads with Cloudinary

---

## 🛠️ Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios
- React Router DOM

Backend:
- Node.js
- Express
- MongoDB + Mongoose
- JSON Web Tokens (JWT)
- Multer + Cloudinary

---

## ⚙️ Installation

Run the following commands:

    git clone https://github.com/shihab-kapadiya-016/inkspire.git && cd inkspire

Install dependencies:

Frontend:
    cd frontend
    npm install

Backend:
    cd backend
    npm install

Set up environment variables:

Create a `.env` file in the backend folder with:

    PORT=5000
    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_jwt_secret
    CLOUDINARY_CLOUD_NAME=your_cloud_name
    CLOUDINARY_API_KEY=your_api_key
    CLOUDINARY_API_SECRET=your_api_secret

Run the app:

Backend:
    cd backend
    npm run dev

Frontend:
    cd frontend
    npm run dev

The app will run at: http://localhost:5173
